## Try It Out in 30 Seconds!

### 1. Open the App
Simply open `index.html` in your browser or visit the live deployment URL.

### 2. First Time Setup (1 minute)
When you first launch:
- Enter your business name
- Add your email and phone
- Optionally upload your logo
- Click "Complete Setup"

### 3. Create Your First Form (2 minutes)

**Use a Template (Fastest):**
1. Click "Create New Form"
2. Choose a template (e.g., "Contact Information Form")
3. Customize if needed
4. Click "Save Template"

**Or Build from Scratch:**
1. Click "Create New Form"
2. Click "Start from Scratch"
3. Add sections and questions
4. Save your template

### 4. Fill Out a Form (1 minute)
1. From Dashboard, click "Fill Form"
2. Select your template
3. Fill it out (or have a client fill it)
4. Add signature if the form has one
5. Submit!

### 5. Export as PDF
1. Go to "View Responses"
2. Find your submitted form
3. Click "Export PDF"
4. Save or print!

## Common Tasks

### Print Blank Forms
1. Go to "My Templates"
2. Click "Export PDF" next to any form
3. Print the PDF

### Share Forms with Clients
Hand your device to the client, or:
1. Export blank PDF
2. Email or print it
3. Client fills it out on paper
4. You can manually enter responses later

### Backup Your Data
1. Go to "View Responses"
2. Click "Export All Data"
3. Save the JSON file
4. Keep it safe!

### Restore from Backup
1. Go to "View Responses"
2. Click "Import Data"
3. Select your JSON backup file
4. Done!

## Troubleshooting

### "App won't install as PWA"
- Use HTTPS (not HTTP)
- Try Chrome or Edge browser
- Check that you're not in Incognito mode

### "Forms not saving"
- Check browser storage settings
- Ensure you clicked "Save" not just "Preview"
- Check browser console for errors

### "Signature not working"
- Try using a mouse or finger (touch screen)
- Clear the signature and try again
- Make sure JavaScript is enabled

### "PDF looks wrong"
- Try a different PDF viewer
- Check if all form fields are filled
- Export again

### "Lost my data!"
- Check if you're using the same browser
- Look for backup JSON files
- IndexedDB data is browser-specific
